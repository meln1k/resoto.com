Resoto by Some Engineering

Find leaky resources, manage quota limits, detect drift, and clean up!

Resource Graph

Resoto indexes resources, captures dependencies, and maps out your infrastructure in a human-friendly graph view with metrics for each resource.

Search and Automation

Developers and SREs can search the graph with a query language, and create alerting and clean-up workflows.

Metrics

Data can be aggregated and exported to a time series database such as Prometheus!
